# Markov Models & Stochastic Processes

A collection of probabilistic models and stochastic processes implemented from scratch in Python, accompanied by mathematical derivations, visualizations, and practical applications.

---

## Overview

This repository explores the theory and implementation of stochastic processes used in Machine Learning, Statistics, Finance, Signal Processing, and Natural Language Processing.

The primary focus is understanding probabilistic sequence modelling, Markovian systems, hidden state estimation, and continuous-time stochastic processes through mathematical derivations and low-level implementations.

---
